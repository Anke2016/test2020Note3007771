U1 &PURPOSE&                                                               
AS When note corrections for objects in ABAP for SAP HANA applications are 
   implemented, it may not be possible to activate SAP HANA repository     
   objects (LIMU HOTO) and packages (LIMU HOTP):                           
*  If this is the case, you can use the program to delete the inactive     
   version of the objects in ABAP.                                         
*  Note: Note that you should use this program only for troubleshooting    
   purposes. More information can be found in SAP Note 2633397 (ht         
=  tps://launchpad.support.sap.com/#/notes/2633397).                       
U1 &ACTIVITIES&                                                            
AS Enter the name of the transport of the object in question or select it  
   using input help. Run the program.
